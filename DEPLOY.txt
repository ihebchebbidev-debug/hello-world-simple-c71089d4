### To Deploy:
```powershell
git push origin main
```

That's it! Vercel will:
1. Clone your repository
2. Run `npm run build`
3. Deploy `api/index.ts` as a serverless function
4. Serve your app

Your app will be live within minutes at your Vercel URL.
